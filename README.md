# 💱 Currency Converter

🔗 **Live Demo:** https://anushka1423.github.io/Currency-convertor/

---

## 📌 Overview

A responsive and feature-rich currency converter web application that fetches real-time exchange rates using an API. Designed with a modern UI and enhanced user experience.

---

## 🚀 Features

* 🌍 Real-time currency conversion
* 🔄 Swap currencies instantly
* 🌙 Dark mode toggle
* ⚡ Fast and responsive UI
* 🎯 Clean and user-friendly interface

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* REST API (Exchange Rates)

## ⚙️ How It Works

1. User selects currencies
2. Enters amount
3. App fetches real-time exchange rates
4. Displays converted value instantly
